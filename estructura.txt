NOTAS DE ARQUITECTURA

Flujo de consulta:
  Usuario → context-orchestrator → [liquidity, events, price-behavior, macro] → synthesis → Usuario

Prioridad de precios:
  1. TradingView MCP (cuando TV Desktop está abierto con --remote-debugging-port=9222)
  2. OANDA v20 API (EURUSD, GBPUSD)
  3. Alpha Vantage (fallback general)
  DXY: solo disponible via TV (TVC:DXY) o scraping

Cobertura de instrumentos:
  - DXY (US Dollar Index)
  - EURUSD
  - GBPUSD
