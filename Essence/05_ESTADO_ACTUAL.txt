================================================================================
PROBLEMAS CONOCIDOS
================================================================================

- quote_get del TV MCP ignora el parámetro symbol — siempre devuelve datos
  del chart activo. Workaround: tab switching (D008)
- ForexFactory calendar scrape retorna 403 (Cloudflare bloquea).
  Workaround: eventos se cargan vía calendario del sistema o input manual.
- El calendario tiene huecos de datos para algunas semanas.
  El trader debe verificar externamente eventos de alto impacto.
